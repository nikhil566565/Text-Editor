# 📝 Text Editor (Python)
---

## 🚀 Overview

This is a simple yet powerful **Text Editor application built using Python and Tkinter**.
It supports essential text editing operations along with useful features like **Unlimited Undo/Redo** and **Dark Mode** for a better user experience.

This project was developed to explore **Python GUI development** and strengthen problem-solving skills.

---

## ✨ Features

* ✂️ Cut, Copy, Paste
* ↩️ Undo & Redo (Unlimited)
* 📂 File Operations:

  * New File
  * Open File
  * Save File
* 🌙 Dark Mode (Night Mode)
* ⚡ Lightweight and fast

---

## 🛠️ Tech Stack

* **Python 3**
* **Tkinter (GUI Toolkit)**

---

## 📋 Requirements

Make sure you have the following installed:

* Python 3.x
* Pip
* Tkinter (comes pre-installed with Python)

---

## ▶️ How to Run

```bash
# Clone the repository
git clone https://github.com/your-username/text-editor.git

# Navigate to the project folder
cd text-editor

# Run the application
python main.py
```
